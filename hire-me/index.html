Aurora Films Website

Project

Create new website with easy-to-use CMS using client-supplied content.

Responsibilities

Design and Development Lead
Wireframes, design gut-check, element collage, front-end design, front-end development, presentations

Process

Using Aurora's first content draft, I created a set of wireframes to organize the content and guide the client's content creation.

Aurora wanted a new look for their site but gave no direction. Instead of guessing what they like, I created presented a design gut check to two of the stakeholders, using screen shots from various websites to kick-start the design conversation.

I created an element collage to show the direction I thought they were looking for based upon the conversation we had.

Although I had never used WordPress, I chose to use it for this project because of it's reputation for ease of use — for both the developer and the content author. With help from the dev team, I was able to learn enough PHP to create a new theme and several custom templates.

JLG Product Launch Landing Page

View site →

Project

We were tasked with creating a landing page for a new product launch, containing an interactive tour of the product. Knowing that there were several other big announcements on the horizon, we proposed building a reuseable landing page template into their CMS (SiteCore) for simple page creation in the future.

We decided to use a modular approach, and let the content creator decide where the components should be placed on the page, and what type of component (WYSIWYG, Video, Image gallery, 360, etc) should appear.

A key component of these pages were the 360 degree tours of the products. I worked with another developer to research, test, and build the 360s — and make all of the content editeable in the CMS.

The template has allowed us to create at least 15 separate landing pages in half the time it used to take.

Responsibilities

Planning, wireframes, front-end design, front-end development
